How It Works

  • Whisper Transcription

    Audio is processed using Groq's whisper-large-v3 API. We pre-process the transcript locally to remove filler words and assess quality before sending to the intelligence pipeline.

  • Stage 1: Fast Analysis

    A high-speed LLaMA 3.1 call (~300 tokens) maps the lecture. It identifies the domain (e.g., Computer Science, Economics), detects the structural breakdown, and spots explicit emphasis markers used by the professor.

  • Stage 2: Domain-Aware Structuring

    Using Stage 1's blueprint, a second LLaMA 3.1 pass acts as a domain expert. It extracts concepts with confidence ratings, identifies exam signals, and generates active recall flashcards.
