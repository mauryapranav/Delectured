Getting Started

  • Get a Groq API Key

    DeLectured runs entirely in your browser using the Groq API for blazing fast inference. Get a free API key at console.groq.com.

  • Prepare your audio

    Record your lecture. M4A (voice memo), MP3, and WAV files up to 25MB are supported. If your file is larger, use a free compressor tool online.

  • Select Language

    DeLectured natively supports mixed-language processing. If your professor mixes Hindi and English (Hinglish), select the "Hindi / Hinglish" option for specialized notes.

  • Upload & Wait

    Processing typically takes 15-45 seconds depending on lecture length, handled directly via API calls from your browser.

Privacy Architecture

Your audio never touches our servers. DeLectured is a static web application. When you upload a file, it is sent directly from your browser to Groq's secure API endpoints. We have no databases, no user accounts, and no access to your API key or data.

Feature Deep-Dives

The Two-Stage Pipeline

Unlike standard AI tools that simply run "summarize this text" over a transcript, DeLectured operates like a human note-taker analyzing a room.

Stage 1 (Analysis): A fast, cheap inference call (~300 tokens) reads the raw transcript to identify the domain (e.g., Biology, Economics), structure (Intro, Core, Examples), and explicit emphasis markers ("this is very important").

Stage 2 (Structuring): A second, specialized inference call uses Stage 1's blueprint to extract concepts dynamically, knowing exactly what the professor emphasized.

Concept Confidence

Not all topics are covered equally. DeLectured plots a visual confidence map: ●●● means a concept was deeply explained, ●○○ means it was only briefly mentioned. This tells you exactly what you need to look up in the textbook.

Exam Signals

A hybrid regex + AI system scans the transcript for explicit and implicit exam hints ("remember this for finals", "यह exam में आएगा"). These are pulled to the very top of your notes.

FAQ

What if my recording is longer than 1 hour?

To prevent API timeouts, DeLectured currently limits analysis to the first ~6000 words. For 2-hour seminars, we recommend splitting the audio file.

Why do I need my own API key?

Because DeLectured has no backend and is completely free. Providing your own Groq key allows you to use their extremely fast inference models at no cost to you (within their generous free tier).

Can I save my notes?

Yes. Use the "Download .TXT" or "Print PDF" buttons at the bottom of the notes. Because there is no database, refreshing the page will clear your current notes.
